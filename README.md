# React Scaffold
包含基础的React 项目结构

# Feature
- ES6+的支持,使用Babel转换
- 开发模式 和 产品模式
- 开发模式热加载支持
- 方便扩展, 同时加入了less, postcss, redux 等loader

# Setup
- git clone https://git.coding.net/starriv/react-scaffold.git
- npm install

# Script
默认为开发模式
- dev npm run dev && 打开浏览器 http://127.0.0.1:3002
- prod npm run build
